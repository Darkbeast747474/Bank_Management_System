# Steps to Run the Bank Management System

1. **Prerequisites**
    - Ensure you have Python installed on your system
    - Have all required dependencies installed

2. **Clone the Repository**
    ```bash
    git clone https://github.com/Darkbeast747474/Bank_Management_System.git
    cd Bank_Management_System
    ```

3. **Install Dependencies**
    ```bash
    pip install -r requirements.txt
    ```

4. **Run The Database Queries**

    - All the Queries Are in File : ```db.sql```

5. **Run the Application**
    ```bash
    python main.py
    ```

6. **Using the System**
    - Use The Username ```admin``` and Password ```admin123```
